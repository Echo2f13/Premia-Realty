## üé® Social Media Image Specifications

### Open Graph Images (Facebook, LinkedIn)
- **Dimensions**: 1200 x 630 pixels
- **Format**: JPG or PNG
- **Max File Size**: 8 MB (recommended under 1 MB)
- **Aspect Ratio**: 1.91:1

### Twitter Card Images
- **Dimensions**: 1200 x 628 pixels (or 2:1 ratio)
- **Format**: JPG, PNG, WEBP, or GIF
- **Max File Size**: 5 MB
- **Aspect Ratio**: 2:1

### Design Tips:
1. Include your logo prominently
2. Use high-quality property images
3. Add text overlay with key message
4. Maintain brand colors and fonts
5. Ensure text is readable on mobile
6. Test on both light and dark backgrounds

## ‚úÖ Testing Your Meta Tags

### Before Going Live:

1. **Facebook Sharing Debugger**
   - URL: https://developers.facebook.com/tools/debug/
   - Enter your page URL
   - Click "Scrape Again" to refresh cache
   - Check image preview, title, and description

2. **Twitter Card Validator**
   - URL: https://cards-dev.twitter.com/validator
   - Enter your page URL
   - Preview how card will look
   - Fix any warnings

3. **LinkedIn Post Inspector**
   - URL: https://www.linkedin.com/post-inspector/
   - Enter your page URL
   - Preview post appearance
   - Clear cache if needed

4. **Google Rich Results Test**
   - URL: https://search.google.com/test/rich-results
   - Enter your page URL
   - Check structured data validity
   - Fix any errors
